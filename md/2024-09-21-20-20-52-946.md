# Q
2016N66A 诊断胃食管反流病最准确的是
A.食管吞钡X线检查
B.食管测压
C.24h食管pH监测
D.胃镜

# A
D
# D
```ad-note
title:课本出处
collapse:false
（P348）“胃镜 是诊断RE最准确的方法，并能判断RE的严重程度和有无并发症”（D对）。
```

```ad-summary
title:解析
collapse:false
胃镜（D对）是诊断RE（反流性食管炎）最准确的方法，并能判断其严重程度和有无并发症，结合活检可与其他原因引起的食管炎和其他食管病变作鉴别。虽然食管吞钡X线检查（A错）、食管测压（B错）、24h食管pH监测（C错）都对本病的诊断有帮助，但都不是最准确的检查方法。
```
