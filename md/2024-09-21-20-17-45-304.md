# Q
2021N47A 一般不会出现在急性心肌炎病人的临床表现的是
A.心脏压塞
B.猝死
C.急性肺水肿
D.酷似急性心肌梗死

# A
A
# D
```ad-note
title:课本出处
collapse:false
（P270）“病毒性心肌炎病人临床表现取决于病变的广泛程度与部位，轻者可完全没有症状，重者甚至出现心源性休克及猝死（B对）。多数病人发病前1～3周有病毒感染前驱症状，如发热、全身倦怠感和肌肉酸痛，或恶心、呕吐等消化道症状。随后可以有心悸、胸痛（D对）、呼吸困难、水肿（C对），甚至晕厥、猝死”。（P305）“3.心脏压塞　短期内出现大量心包积液可引起急性心脏压塞，表现为窦性心动过速、血压下降、脉压变小和静脉压明显升高”（A错，为本题正确答案）。
```

```ad-summary
title:解析
collapse:false
各种病因心包炎均可能伴有心包积液，心脏压塞（A错，为本题正确答案）是短期内出现大量心包积液所导致的，表现为临床特征为Beck's三联征：血压低、心音弱、颈静脉怒张。急性心肌炎的临床表现取决于病变的广泛程度与部位，轻者几乎无症状，重者可导致猝死（B对），多数患者发病前可有发热、全身酸痛、咽痛、腹泻等症状，常诉胸闷、心前区疼痛、心悸、乏力、恶心、头晕等（酷似急性心肌梗死）（D对），疾病发展迅速的患者会出现心力衰竭、急性肺水肿（C对）或心源性休克。
```
