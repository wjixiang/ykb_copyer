# Q
1999N132C Crigler-Najjar综合征
A.胆红素摄取障碍
B.胆红素结合障碍
C.两者均有
D.两者均无

# A
B
# D
```ad-note
title:课本出处
collapse:false
（九版诊断学P38）“Crigler-Najjar综合征：系由肝细胞缺乏葡萄糖醛酸转移酶，致UCB不能形成CB，导致血中UCB增多而出现黄疸”（B对）。
```

```ad-summary
title:解析
collapse:false
Crigler-najjar综合征（CNS）又称为先天性葡萄糖醛酸转移酶缺乏症，病因为由于肝脏缺乏葡萄糖醛酸转移酶，而不能催化非结合胆红素与葡萄糖醛酸结合，形成结合胆红素（B对），但胆红素摄取正常（A错）。
```
