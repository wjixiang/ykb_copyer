# Q
2015N62A 我国高血压人群个体特点是
A.盐敏感
B.高血糖
C.高血脂
D.高肾素

# A
A
# D
```ad-note
title:课本出处
collapse:false
（P248）“不同地区人群血压水平和高血压患病率与钠盐平均摄入量显著正相关…摄盐过多导致血压升高主要见于对盐敏感人群”（A对）。
```

```ad-summary
title:解析
collapse:false
高血压的发病主要与遗传、饮食、精神应激、吸烟、体重及药物等有关。我国不同地区的人群高血压患病率与当地钠盐平均摄入量存在显著正相关，并且在我国高血压人群中，盐敏感（A对）者大约占25%～42%。高血糖（B错）、高血脂（C错）是我国高血压患者的心血管高危因素，而高肾素（D错）是我国高血压患者的发病机制之一。
```
