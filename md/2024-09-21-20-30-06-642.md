# Q
1992N113C 肾小球肾炎
A.尿沉渣镜检有大量变形红细胞
B.尿沉渣镜检有多个嗜酸性粒细胞
C.两者皆有
D.两者皆无

# A
A
# D
```ad-note
title:课本出处
collapse:false
（P465）“2.血尿…①新鲜尿沉渣相差显微镜检查。变形红细胞尿为肾小球源性（A对），均一形态正常红细胞尿为非肾小球源性”。
```

```ad-summary
title:解析
collapse:false
肾小球源性血尿产生的主要原因为GBM断裂，红细胞通过该裂缝时受血管内压力挤压受损，受损的红细胞后通过肾小管各段又受不同渗透压和pH作用呈现变形红细胞（A对）血尿，红细胞容积变小，甚至破裂。尿沉渣镜检有多个嗜酸性粒细胞（B错）见于过敏性疾病、皮肤病、寄生虫病、某些血液病等。
```
