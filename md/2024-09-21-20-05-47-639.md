# Q
2024N45A 男性，56岁。低热、盗汗伴咳嗽、咳痰2月余。既往吸烟30年，20支/日。查体：体型消瘦，浅表淋巴结无肿大，双肺听诊无异常。血WBC8.0×10⁹/L，N59%，肺部X线片示“左上肺密度不均匀片状影”。最可能的诊断是
A.肺结核
B.肺炎
C.肺脓肿
D.肺癌

# A
A
# D
```ad-note
title:课本出处
collapse:false
（9版内科学P65）“1.呼吸系统症状　咳嗽、咳痰两周以上或痰中带血是肺结核的常见可疑症状…2.全身症状　发热为最常见症状，多为长期午后潮热，即下午或傍晚开始升高，翌晨降至正常。部分病人有倦怠乏力、盗汗、食欲减退和体重减轻等…2.影像学诊断…肺结核病影像特点是病变多发生在上叶的尖后段、下叶的背段和后基底段，呈多态性，即浸润、增殖、干酪、纤维钙化病变可同时存在，密度不均匀、边缘较清楚和病变变化较慢，易形成空洞和播散病灶”（9版诊断学P244）“1.淋巴细胞增多…（1）感染性疾病…也可见于百日咳杆菌、结核分枝杆菌、布鲁菌、梅毒螺旋体、弓形体等的感染”（A对）。
```

```ad-summary
title:解析
collapse:false
中年男性患者，低热、盗汗伴咳嗽、咳痰2月余（迁延不愈咳嗽咳痰、长期低热，提示结核杆菌感染），吸烟30年，20支/日（吸烟易诱发气道炎症，降低呼吸系统免疫力），查体：体型消瘦（结核病全身表现），浅表淋巴结无肿大，肺部听诊无异常（暂时排除肺部其他感染性疾病），WBC8.0×10⁹/L，N59%（暂排除其他急性感染），肺部X线示“左上肺密度不均匀片状影”（肺结核可有影像学表现），根据患者病史、临床表现和体征及影像学检查结果，最可能的诊断是肺结核（A对）。肺结核的诊断依赖病原学检查，但病史和临床表现仍然是诊断的基础，凡遇下列情况者应高度警惕结核病的可能性：①反复发作或迁延不愈的咳嗽咳痰或呼吸道感染经抗生素治疗3～4周仍无改善；②痰中带血或咯血；③长期低热或所谓“发热待查”；④体检肩胛间区有湿啰音或局限性哮鸣音；⑤有结核病诱因或好发因素，尤其是糖尿病、免疫抑制性疾病和接受激素或免疫抑制剂治疗者；⑥有关节疼痛和皮肤结节性红斑、滤泡性结膜角膜炎等过敏反应性表现；⑦有渗出性胸膜炎、肛瘘、长期淋巴结肿大既往史以及婴幼儿和儿童有家庭开放性肺结核密切接触史者。肺炎（B错）与肺结核的鉴别的关键证据是病原学检测，临床表现方面如细菌性肺炎一般起病较急，高热寒战，伴有胸痛，血白细胞总数及中性粒细胞增多，抗菌药物治疗有效，可资鉴别。肺脓肿（C错）影像学可见空洞，多见于肺下叶（结核空洞多为上叶，主要与厌氧菌和结核杆菌对氧气的需求有关），脓肿周围的炎症浸润较严重，空洞内常有液平面，肺脓肿起病较急、高热、脓臭痰，但痰中无结核菌，血白细胞总数及中性粒细胞增多，抗生素治疗有效。肺癌（D错）多表现为刺激性咳嗽、胸痛和进行性消瘦，中央型肺癌可有痰中带血，影像学肺癌病灶边缘常见切迹、毛刺，胸部CT有助于二者鉴别，结合痰结核菌、脱落细胞检查及纤维支气管镜检查和活检等能及时鉴别，需要注意的是肺癌和肺结核可有并存。
```
