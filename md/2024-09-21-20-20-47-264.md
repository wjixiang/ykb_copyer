# Q
2002N132C Barrett溃疡是指
A.鳞状上皮被柱状上皮取代
B.发生消化道溃疡
C.两者均是
D.两者均非

# A
C
# D
```ad-note
title:课本出处
collapse:false
九版内科学未明确说明。
```

```ad-summary
title:解析
collapse:false
Barrett食管是指食管的鳞状上皮出现单层柱状上皮化生（A对），多发生于胃食管连接处的齿状线近端。Barrett溃疡是指在Barrett食管基础上发生的消化性溃疡（B对）。本题AB均对，正确答案为C。
```
