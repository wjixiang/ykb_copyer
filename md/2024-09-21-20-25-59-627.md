# Q
2017N156X 溃疡性结肠炎的并发症可有
A.中毒性巨结肠
B.癌变
C.肠出血
D.肠穿孔

# A
ABCD
# D
```ad-note
title:课本出处
collapse:false
（P374）“【并发症】1.中毒性巨结肠（A对）…本并发症易引起急性肠穿孔…2.癌变（B对）…3.其他并发症 结肠大出血（C对）发生率约3%；肠穿孔（D对）多与中毒巨结肠有关”。
```

```ad-summary
title:解析
collapse:false
溃疡性结肠炎的并发症最常见的为中毒性巨结肠（A对）（5%）和直肠结肠癌变（B对）（多见于病程漫长者），此外还有结肠出血（C对）（3%）和肠穿孔（D对）（并发于中毒性巨结肠）。
```
