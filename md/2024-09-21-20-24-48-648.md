# Q
1997N130C 治疗幽门梗阻用
A.阿托品
B.H₂受体阻滞剂
C.两者均可
D.两者均不可

# A
B
# D
```ad-note
title:课本出处
collapse:false
九版内科学未明确说明。
```

```ad-summary
title:解析
collapse:false
抗胆碱药物（阿托品）能松弛平滑肌，延缓胃排空，引起胃窦部潴留，因此对幽门梗阻者不适用（A错）。幽门梗阻的患者一般都有较长的溃疡病史，故抑酸药H₂受体拮抗剂（H₂RA）可用于幽门梗阻的治疗（B对）。
```
