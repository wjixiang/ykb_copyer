# Q
1993N50A 心绞痛发作时可出现
A.体温升高
B.血沉增快
C.血清心肌酶谱增高
D.动脉血压增高
E.以上都不正确

# A
D
# D
```ad-note
title:课本出处
collapse:false
（P219）“（二）体征 心绞痛发作时常见心率增快、血压升高（D对）、表情焦虑、皮肤冷或出汗，有时出现第四或第三心音奔马律”。
```

```ad-summary
title:解析
collapse:false
心绞痛发作时由于疼痛刺激，交感神经兴奋，可见患者心率增快、血压升高（D对）、表情焦虑、皮肤冷或出汗，有时出现第四或第三心音奔马律。体温升高（A错），血沉增快（B错），血清心肌酶谱增高（C错）是急性心肌梗死的表现，而单纯的心绞痛不会出现上述表现。
```
