# Q
1998N152X 自发性气胸的常见病因有
A.肺大疱
B.肺结核
C.肺癌
D.用力咳嗽、屏气、喷嚏

# A
ABC
# D
```ad-note
title:课本出处
collapse:false
（P120）“继发性自发性气胸多见于有基础肺部病变者，由于病变引起细支气管不完全阻塞，形成肺大疱破裂。如肺结核、慢阻肺、肺癌、肺脓肿、肺尘埃沉着症及淋巴管平滑肌瘤病等”（P120）“起病前有的病人可能有持重物、屏气、剧烈体力活动等诱因，但大多数病人在正常活动或安静休息时发生，偶有在睡眠中发病者。大多数起病急骤，病人突感一侧胸痛，针刺样或刀割样，持续时间短暂，继之胸闷和呼吸困难，可伴有刺激性咳嗽，系气体刺激胸膜所致”（P120）“体征…呼吸运动与触觉语颤减弱，叩诊过清音或鼓音，心或肝浊音界缩小或消失，听诊呼吸音减弱或消失”（A对）。
```

```ad-summary
title:解析
collapse:false
患者男性，40岁，长期吸烟史，慢性咳嗽咳痰史，考虑有COPD（D错）基础疾病（继发性气胸常见于有基础肺部病变者），在此基础上，突发左侧胸痛1天，呈针刺样疼痛，向左肩部放射，伴有胸闷及气短，干咳（气胸常见的临床症状），无发热。左下肺叩诊鼓音，左下肺呼吸音明显降低，右肺散在少量干啰音，心界叩诊不清（气胸的临床体征），最可能的疾病为左侧气胸（A对）。肺栓塞（P100）（B错）常有房颤、长期卧床、下肢血栓形成等病史，表现为突发胸痛、呼吸困难、咯血、晕厥、心动过速、颈静脉充盈或搏动，肺动脉瓣区第二心音亢进或分裂，三尖瓣区收缩期杂音；急性心肌梗死（P235-P236）（C错）常表现为心前区疼痛，有濒死感，可出现发热、心动过速、恶心呕吐，大汗淋漓，服用硝酸甘油不能缓解等症状；肺大疱（E错）一般没有症状和阳性体征。
```
