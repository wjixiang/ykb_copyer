# Q
2001N129C 肾炎综合征表现有
A.血尿
B.高血压
C.二者都有
D.二者都无

# A
C
# D
```ad-note
title:课本出处
collapse:false
（P460）“2.肾炎综合征 以肾小球源性血尿（A对）为主要特征，常伴有蛋白尿。可有水肿和高血压（B对）和（或）肾功能损害”。本题AB均对，正确答案为C。
```

```ad-summary
title:解析
collapse:false
肾炎综合征是指血尿（A对）、蛋白尿、高血压（B对）、水肿为表现的一组综合征。本题AB均对，正确答案为C。
```
