# Q
2015N142B 对结核分枝杆菌B群作用最强的药物是
A.异烟肼
B.利福平
C.吡嗪酰胺
D.乙胺丁醇

# A
C
# D
```ad-note
title:课本出处
collapse:false
（P69）“对B菌群依次为吡嗪酰胺＞利福平＞异烟肼”（C对）。
```

```ad-summary
title:解析
collapse:false
结核分枝杆菌根据其代谢状态分为A、B、C、D4个菌群。抗结核药物对不同菌群的作用各异。A菌群作用强弱依次为异烟肼＞链霉素＞利福平＞乙胺丁醇（D错）；B菌群依次为吡嗪酰胺（C对）＞利福平（B错）＞异烟肼（A错）；C菌群依次为利福平＞异烟肼；抗结核药物对D菌群无作用。
```
