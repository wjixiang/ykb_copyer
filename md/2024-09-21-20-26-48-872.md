# Q
2008N171X 下列肝硬化的临床类型中，发现肝脏肿大的有
A.肝炎后肝硬化
B.淤血性肝硬化
C.酒精性肝硬化
D.原发性胆汁性肝硬化

# A
BCD
# D
```ad-note
title:课本出处
collapse:false
九版内科学未明确说明。
```

```ad-summary
title:解析
collapse:false
肝炎后肝硬化早期肝肿大，晚期肝萎缩变小（A），A选项存在争议。淤血性肝硬化（B对）由于肝淤血，引起肝脏肿大。酒精性肝硬化（C对）常伴黄疸，肝大并有触痛。原发性胆汁性肝硬化（D对）由于胆汁淤积，肝中度到重度肿大，常在肋下4～10cm，质硬，表面光滑，压痛不明显。
```
