# Q
2000N105B 胆红素生成过多见于
A.Gilbert综合征
B.无效造血
C.Dubin-Johnson综合征
D.Crigler-Najjar综合征
E.Rotor综合征

# A
C
# D
```ad-note
title:课本出处
collapse:false
（九版诊断学P38）“Glbert综合征（A错）：系由肝细胞摄取UCB功能障碍及微粒体内葡萄糖醛酸转移酶不足，致血中UCB增高而出现黄疸”。（九版诊断学P38）“Dubin-Johnson综合征（C对）：系由肝细胞对CB及某些阴离子（如靛氰绿、X线造影剂）向毛细胆管排泄发生障碍，致血清CB增加而发生的黄疸”。（九版诊断学P38）“Crigler-Najjar综合征（D错）：系由肝细胞缺乏葡萄糖醛酸转移酶，致UCB不能形成CB，导致血中UCB增多而出现黄疸”。（九版诊断学P38）“Rotor综合征（E错）：系由肝细胞摄取UCB和排泄CB存在先天性缺陷致血中胆红素增高而出现黄疸”。
```

```ad-summary
title:解析
collapse:false
Dubin-Johnson综合征（C对）是临床上少见的一种遗传性非溶血性高胆红素血症（又称家族性高胆红素血症），是由于肝细胞的先天缺陷对CB（结合型胆红素）及某些阴离子向毛细胆管排泄发生障碍，致血清中CB增加而出现黄疸，而胆红素的摄取和结合正常。无效造血（B错）在骨髓内红细胞分裂成熟的过程中，由于某种原因使其在成熟和进入外周循环之前就被破坏、死亡，称之为无效造血，这种途径导致血红蛋白释放、破坏增加，随之胆红素生成增加，而与肝细胞排泄胆红素功能障碍无关。Gilbert综合征（A错）又称体质性肝功能不良性黄疸，一种遗传性非结合胆红素血症，是肝细胞摄取游离胆红素障碍及微粒体内葡萄糖醛酸转移酶不足所致。Crigler-najjar综合征（D错）病因为先天性葡萄糖醛酸转移酶缺乏，导致非结合胆红素转化障碍，而与结合胆红素排泄障碍无关。发生于新生儿和婴幼儿的遗传性高胆红素血症，又称先天性葡萄糖醛酰转移酶缺乏症、伴有胆红素脑病（核黄疸）的先天性非溶血性黄疸等。Rotor综合征（E错）的病因是肝细胞摄取游离胆红和排泄结合胆红素均有先天性缺陷，致血中结合胆红素增高为主。
```
