# Q
1996N130C 克雷伯杆菌肺炎
A.咳砖红色痰
B.红霉素治疗有效
C.两者均是
D.两者均不是

# A
A
# D
```ad-note
title:课本出处
collapse:false
（P44）“表2-6-1 常见肺炎的症状、体征和X线特征”（A对）。
```

```ad-summary
title:解析
collapse:false
克雷伯杆菌肺炎痰液特点为砖红色胶冻状痰（A对），红霉素对支原体、衣原体、军团菌肺炎有效，而肺炎克雷伯杆菌治疗应首选氨基糖苷类抗生素+半合成青霉素（B错）。
```
