# Q
1999N130C 单纯右心室梗死
A.肺动脉压增高
B.肺毛细血管楔压增高
C.两者均有
D.两者均无

# A
D
# D
```ad-note
title:课本出处
collapse:false
（P235）“右心室梗死在MI患者中少见，其主要病理生理改变是急性右心衰竭的血流动力学变化，右心房压力增高，高于左心室舒张末期压，心排血量减低，血压下降”。
```

```ad-summary
title:解析
collapse:false
单纯右心室梗死在MI（心肌梗死）患者中少见，因右心室射血能力下降，血液从右房流入右室受阻，出现右房高压，导致回心血量减少，中心静脉压增高。肺动脉高压（A错）、肺毛细血管楔压增高（B错）是由左心衰导致肺淤血而引起（本题AB均错，正确答案为D）。
```
