# Q
2009N170X 导致肺心病Ⅱ型呼吸衰竭患者发生呼吸性酸中毒合并代谢性碱中毒的常见病因有
A.使用呼吸兴奋剂
B.补碱
C.过度机械通气
D.利尿

# A
BCD
# D
```ad-note
title:课本出处
collapse:false
（P112）“但是利尿药应用后易出现低钾、低氯性碱中毒”（D对）。（P141）“使用原则：①必须保持气道通畅，否则会促发呼吸肌疲劳，加重CO₂潴留；②脑缺氧、脑水肿未纠正而出现频繁抽搐者慎用；③病人的呼吸肌功能基本正常；④不可突然停药。主要适用于以中枢抑制为主、通气量不足引起的呼吸衰竭，不宜用于以肺换气功能障碍为主所致的呼吸衰竭”（P138）“慢性呼吸衰竭时因CO₂潴留发展缓慢，肾脏可通过减少HCO₃⁻的排出来维持pH的恒定。但当体内CO₂长期增高时，HCO₃⁻也维持在较高水平，导致呼吸性酸中毒合并代谢性碱中毒，此时pH可处于正常范围，称为代偿性呼吸酸中毒合并代谢性碱中毒”（P142）“慢性呼吸衰竭常有CO₂潴留，导致呼吸性酸中毒。呼吸性酸中毒的发生多为慢性过程，机体常通过增加碱储备来代偿，以维持pH于相对正常水平。当以机械通气等方法较为迅速地纠正呼吸性酸中毒时，原已增加的碱储备会使pH升高，对机体造成严重危害，故在纠正呼吸性酸中毒时，应注意同时纠正潜在的代谢性碱中毒”（C对）。
```

```ad-summary
title:解析
collapse:false
对于Ⅱ型呼吸衰竭患者而言，合理使用呼吸兴奋剂（A错）能增加呼吸肌做功，增加通气量，能改善呼吸性酸中毒，不会发生代谢性碱中毒；在慢性Ⅱ型呼吸衰竭患者，CO₂潴留发展缓慢，肾脏通过减少HCO₃⁻的排出来维持pH的恒定，体内代偿性地会有较高水平的碳酸氢盐，补碱（B对）则使体内碳酸氢盐的含量过高，导致在呼酸的基础上发生代谢性碱中毒，且补碱不容易控制量，也易造成代谢性碱中毒；Ⅱ型呼吸衰竭时，过度机械通气（C对）可迅速改善呼吸性酸中毒（但仍为呼吸性酸中毒），而代偿性增加的碳酸氢盐却不能很快下降，使体内原有的碳酸氢盐维持在一个较高的水平，导致呼吸性酸中毒合并代谢性碱中毒；使用利尿剂（D对）可抑制Na⁺、Cl⁻重吸收，促进K⁺排出，细胞外液中K⁺减少，细胞H⁺-K⁺交换增加，H⁺进入细胞内导致细胞外液碱中毒，故使用利尿剂容易在原有的呼酸基础上发生低钾低氯性碱中毒。
```
