# Q
2017N54A 下列检查结果支持溶血性贫血的是
A.尿中尿胆原排泄减少
B.血清非结合胆红素减少
C.血清结合珠蛋白减少
D.血网织红细胞减少

# A
C
# D
```ad-note
title:课本出处
collapse:false
（P553）“表6-6-1溶血性贫血的筛查试验”（C对）。
```

```ad-summary
title:解析
collapse:false
溶血性贫血时，尿中尿胆原排泄增加（A错），血清非结合胆红素升高（B错），血网织红细胞升高（D错）；各种溶血时，血清结合珠蛋白均有减低，血管内溶血时显著减低，当发生严重血管内溶血时，可监测不出血清结合珠蛋白（C对）。
```
