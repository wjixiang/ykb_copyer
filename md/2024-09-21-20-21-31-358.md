# Q
1994N58A 判断胃酸缺乏最可靠的指标是
A.呕吐物无酸
B.空腹胃液pH为7
C.BAO为0
D.MAO为0
E.空腹胃液中游离酸为0

# A
D
# D
```ad-note
title:课本出处
collapse:false
九版内科学未明确说明。
```

```ad-summary
title:解析
collapse:false
MAO（最大胃酸分泌）（D对）是指给予刺激后连续收集1小时胃液所测泌酸值，若无酸，即MAO为0，是胃酸缺乏最可靠的指标。BAO（基础胃酸分泌）（C错）是指抽取空腹胃液后，再连续收集1小时胃液所测泌酸值，未加刺激剂，若为0，刺激后还可能会有酸分泌，所以不如MAO可靠。呕吐物无酸（A错）、空腹胃液pH为7（B错）和空腹胃液中游离酸为0（E错）的可靠性更差。
```
